## Tecnologias

- ReactJS
- NextJS
- Supabase
- chakra-ui
- TypeScript

## Como rodar o projeto?

Primeiro, execute o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
